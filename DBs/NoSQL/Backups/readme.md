# 📂 Backups
![MongoDB](https://img.shields.io/badge/MongoDB-47A248?style=for-the-badge&logo=mongodb&logoColor=white)  ![NoSQL](https://img.shields.io/badge/NoSQL-FF6F00?style=for-the-badge&logo=apache-couchdb&logoColor=white)

## 📁 Descripción
La carpeta Backups almacena copias de seguridad de la base de datos NoSQL, garantizando la protección y disponibilidad de los datos en caso de fallos, errores o pérdidas. Estas copias permiten restaurar la información de manera rápida y eficiente, minimizando el impacto en el sistema.

Los backups son esenciales para la administración y seguridad de la base de datos, asegurando la continuidad operativa ante eventos inesperados.

### 📌 Subcarpetas

- **Automated**: Contiene copias de seguridad automáticas, generadas según una programación establecida. Estos respaldos se crean regularmente para garantizar la disponibilidad de versiones recientes sin intervención manual.

- **Manual**: Almacena copias de seguridad realizadas manualmente por los administradores de la base de datos. Estas pueden incluir versiones específicas antes de realizar cambios críticos en la estructura o datos almacenados.
