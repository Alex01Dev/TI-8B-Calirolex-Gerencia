# 📂 Full 
![MongoDB](https://img.shields.io/badge/MongoDB-47A248?style=for-the-badge&logo=mongodb&logoColor=white)  ![NoSQL](https://img.shields.io/badge/NoSQL-FF6F00?style=for-the-badge&logo=apache-couchdb&logoColor=white)

## 📁 Descripción
La carpeta **Full**, almacena copias de seguridad completas de la base de datos **NoSQL** que han sido generadas manualmente por los administradores del sistema. Estos respaldos incluyen toda la estructura y los datos, permitiendo la restauración total del sistema en caso de fallos críticos.

## 📌 Características
- ✅ Contiene todas las colecciones, documentos, índices y configuraciones.
- ✅ Garantiza una recuperación completa del sistema ante pérdidas de datos.
- ✅ Se generan manualmente antes de actualizaciones, cambios estructurales o mantenimiento.
- ✅ Ideal para migraciones, auditorías y respaldo antes de modificaciones críticas.
