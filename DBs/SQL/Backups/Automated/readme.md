 
# 📂 Automated  

## 📁 Descripción  

La carpeta **Automated** contiene copias de seguridad generadas automáticamente por el sistema. Estas backups se crean según una programación definida, asegurando la protección continua de los datos sin intervención manual.  
 

### 📌 **Subcarpetas**  

- **Full**: Almacena copias de seguridad completas de la base de datos. Estas backups incluyen todas las tablas, registros, índices y estructuras, permitiendo una restauración completa del sistema en caso de fallos graves.  

- **Partial**: Contiene copias de seguridad parciales, donde solo se respaldan ciertos elementos de la base de datos, como tablas específicas, registros críticos o configuraciones clave. Son útiles para restauraciones más rápidas y específicas.
