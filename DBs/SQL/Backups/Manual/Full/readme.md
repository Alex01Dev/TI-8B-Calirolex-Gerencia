# 📂 Full  
![MySQL](https://img.shields.io/badge/MySQL-005C84?style=for-the-badge&logo=mysql&logoColor=white)  ![SQL](https://img.shields.io/badge/SQL-4479A1?style=for-the-badge&logo=amazon-rds&logoColor=white)  

## 📁 Descripción  

La carpeta **Full** contiene copias de seguridad completas de la base de datos SQL, creadas manualmente por los administradores. Estas backups incluyen toda la estructura, datos, relaciones e índices.  

### 📌 **Características**  

✅ Contiene la base de datos completa en el momento del respaldo.  
✅ Se usa antes de actualizaciones importantes o cambios en la infraestructura.  
✅ Permite restauraciones completas en caso de fallos.  
✅ Se almacena de forma segura según las políticas de la empresa.
