# 📂 Partial  
![MySQL](https://img.shields.io/badge/MySQL-005C84?style=for-the-badge&logo=mysql&logoColor=white)  ![SQL](https://img.shields.io/badge/SQL-4479A1?style=for-the-badge&logo=amazon-rds&logoColor=white)  

## 📁 Descripción  

La carpeta **Partial** almacena copias de seguridad parciales de la base de datos SQL, creadas manualmente. Estas backups incluyen elementos específicos, como tablas individuales, registros críticos o configuraciones esenciales.  

### 📌 **Características**  

✅ Permite respaldar solo partes esenciales de la base de datos.  
✅ Útil para pruebas, migraciones o recuperación de información puntual.  
✅ Se crea manualmente según las necesidades del equipo.  
✅ Facilita una restauración rápida sin afectar toda la base de datos.
