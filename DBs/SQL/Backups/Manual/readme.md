# 📂 Manual  
![MySQL](https://img.shields.io/badge/MySQL-005C84?style=for-the-badge&logo=mysql&logoColor=white)  ![SQL](https://img.shields.io/badge/SQL-4479A1?style=for-the-badge&logo=amazon-rds&logoColor=white)  

## 📁 Descripción  

La carpeta **Manual** contiene copias de seguridad de la base de datos SQL realizadas de manera manual por los administradores del sistema. Estas backups suelen generarse antes de actualizaciones importantes, cambios en la estructura de datos o para cumplir con auditorías y normativas de seguridad.  


### 📌 **Subcarpetas**  

- **Full**: Copias de seguridad completas de la base de datos, incluyendo todas las tablas, registros y configuraciones.  

- **Partial**: Respaldos parciales de elementos específicos de la base de datos, como ciertas tablas o datos críticos.
