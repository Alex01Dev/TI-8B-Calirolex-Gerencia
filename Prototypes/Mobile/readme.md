## 📁 Descripción  

Esta carpeta contiene el código fuente y documentación relacionada con la aplicación móvil del proyecto. La app está desarrollada en **Flutter** con **Dart**, proporcionando una experiencia fluida y nativa en dispositivos Android e iOS.


Esta aplicación móvil permitirá al área de gerencia del gimnasio:

1. **Visualizar métricas clave del gimnasio en tiempo real**
2. **Gestionar la asignación de entrenadores y clases**
3. **Acceder a reportes de rendimiento y estadísticas de usuarios**
4. **Monitorear el uso de equipos e instalaciones**


### 📌 **Subcarpetas**  

- **Assets**: Contiene imágenes, íconos y otros recursos utilizados en la app.
- **Lib**: Código fuente de la aplicación en Flutter.
- **Docs**: Documentación técnica y funcional de la aplicación móvil.
- **Test**: Pruebas unitarias y de integración para asegurar la estabilidad de la app.
