## 📁 Descripción  

Esta carpeta contiene los archivos de hojas de estilo en cascada (CSS) que definen la apariencia y el diseño de la interfaz del **módulo de gerencia del gimnasio**.  

Los estilos en esta carpeta permiten que la aplicación sea visualmente atractiva, accesible y adaptable a diferentes dispositivos mediante **diseño responsivo**. Se incluyen reglas de estilo globales, temas de color, tipografías y personalización de componentes.
