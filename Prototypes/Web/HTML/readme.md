![HTML](https://img.shields.io/badge/HTML-Structure-red?style=for-the-badge&logo=html5&logoColor=white)

## 📁 Descripción  

Esta carpeta contiene los archivos **HTML** que definen la estructura del **módulo de gerencia del gimnasio**. Aquí se encuentran las páginas principales de la aplicación, organizadas en componentes reutilizables que permiten la visualización de información relevante como **reportes de membresías, gráficos de usuarios y rendimiento de cada sucursal**.  

Los archivos HTML trabajan en conjunto con **CSS** y **JavaScript** para proporcionar una experiencia de usuario fluida e interactiva. También se integran con bibliotecas como **Vue.js** para la generación dinámica de contenido.  

### 📌 Contenido  

- **index.html**: Página principal del módulo de gerencia.  
- **dashboard.html**: Interfaz con gráficos y reportes de sucursales.  
- **reportes.html**: Página dedicada a la generación de reportes mensuales y análisis de tendencias.  
- **configuracion.html**: Configuración de parámetros administrativos y ajustes del sistema.
