 # 📂 Sitemap

![Structure](https://img.shields.io/badge/Structure-Sitemap-orange?style=for-the-badge)  

## 📁 Descripción  

Esta carpeta contiene el **mapa del sitio** web del módulo de gerencia del gimnasio. Define la jerarquía y conexión entre las páginas del sistema, facilitando la planificación del flujo de navegación.  

## 📌 Contenido  

- Representación visual de la estructura del sitio  
- Jerarquía de páginas y módulos  
- Explicación de conexiones entre secciones
