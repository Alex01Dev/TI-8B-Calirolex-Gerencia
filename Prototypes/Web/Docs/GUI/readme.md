## 📁 Descripción  

Esta carpeta contiene los elementos de diseño de la **interfaz gráfica de usuario (GUI)** del módulo de **Gerencia** del sistema web en desarrollo. Aquí se incluyen los componentes visuales y la estructura de navegación de la plataforma.

### 📌 **Subcarpetas**  

- **Mockups**: Representaciones visuales detalladas de la interfaz de usuario.
- **Navegational Prototype**: Prototipos interactivos para definir la experiencia de navegación.
- **Sitemap**: Mapa de la estructura del sistema y sus páginas.
- **Sketches**: Bocetos iniciales de la interfaz y disposición de elementos.
- **Wireframes**: Estructuras básicas de las pantallas del sistema.
