## 📁 Descripción  

Esta carpeta contiene los **wireframes** de baja fidelidad para la interfaz del módulo de gerencia del gimnasio. Representan la estructura básica de las pantallas sin detalles gráficos avanzados.  

## 📌 Contenido  

- Esquemas de la disposición de elementos  
- Distribución de contenido en cada pantalla
